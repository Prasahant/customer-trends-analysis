# 📌 Project Overview

Customer Trends Analysis is an end-to-end data analytics project that analyzes customer shopping behavior using **Python**, **SQL**, and **Power BI**. The project performs data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive dashboard creation to uncover purchasing trends, customer preferences, seasonal demand, and revenue insights.

The goal of this project is to help businesses make **data-driven decisions** by understanding customer demographics, shopping habits, payment preferences, and spending patterns.

---

# 🎯 Objectives

- Clean and preprocess raw customer data
- Perform Exploratory Data Analysis (EDA)
- Analyze customer behavior using SQL
- Create an interactive Power BI dashboard
- Generate business recommendations
- Discover hidden trends from customer purchasing behavior
